# Project-03 Commit Convention

Use this format for every commit.

## Commit Title Format
<type>(<scope>): <summary> [<task-id>] [<date>]

Example:
docs(iteration): add task board and daily log [T-2026-04-12-005] [2026-04-12]

## Commit Body Template
Tasks:
- <task-id>

Date:
- <YYYY-MM-DD>

What was done:
- <bullet 1>
- <bullet 2>

Why:
- <reason>

## Allowed Types
- feat
- fix
- docs
- refactor
- test
- chore

## Required Rules
1. At least one Task ID is mandatory.
2. Date tag is mandatory.
3. Summary must be imperative and concise.
4. If task is completed, update 03-task-board.md and 04-daily-work-log.md in the same commit.
